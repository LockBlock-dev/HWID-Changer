## How to use

-   Download the release or build the tool

-   Run it

-   Choose from the available options

## Build instructions - DotNet

-   Download [DotNet](https://dotnet.microsoft.com)

-   Download the project or clone it

-   Run `dotnet build` _(ignore warnings)_

-   Find the result inside `HWID-Changer\bin\Debug\netX.X` _(your dotnet version)_

## Build instructions - Visual Studio

-   Download [DotNet](https://dotnet.microsoft.com)

-   Download the project or clone it

-   Open the solution inside Visual Studio

-   In the menu bar, choose Build and build the solution

-   Find the result inside `HWID-Changer\bin\Debug\netX.X` _(your dotnet version)_
